# Autonivelante

Autonivelante es una empresa Chilena que en base a sus productos de ultima tecnología logran nivelar pisos y dar acabados novedosos ayudando a la  industria y manufactura nacional.

## Desarrollo

El proyecto fue desarrollado con [Next.js](https://nextjs.org/) para el lado del servidor y [React](https://reactjs.org/) para el lado del cliente.

## Despliegue

El proyecto fue desplegado en [cPanel](https://cpanel.net/) utilizando el framework [Next.js](https://nextjs.org/).

## Licencia

El proyecto se encuentra bajo la licencia [MIT](https://choosealicense.com/licenses/mit/).

## Instalación

Usa los paquetes npm para instalar el proyecto y sus dependencias.

```bash
npm install
```
